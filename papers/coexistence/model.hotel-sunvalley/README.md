# Hotel Sunvalley Nasu | ホテルサンバレー那須

### Coexistence Model / 共存モデル

Regional coexistence model based on a participatory Christmas hymn project held at Hotel Sunvalley Nasu, Japan.

ホテルサンバレー那須において実施された、地域参加型クリスマス聖歌企画を中心とした地域共存モデル研究。

---

## This Model / 本モデル

This model examines how existing regional elements can be reconnected to create new cultural and social experiences.

本モデルでは、地域社会に既に存在している要素同士が再接続されることで生まれる、新たな文化体験と共同体構造を研究対象としている。

---

## Main Elements / 主な構成要素

- local residents / 地域住民
- tourists / 観光客
- elderly participation / 高齢者参加
- music and choir / 音楽と合唱
- sacred church space / 教会空間
- emotional resonance / 感情共鳴
- information design / 情報設計

---

## Main Topics / 主な研究テーマ

- participatory tourism / 参加型観光
- shared-time experience / 共に時間を過ごす体験
- church space reactivation / 教会空間の再活性化
- emotional resonance / 感情共鳴
- cultural participation / 文化参加
- social connection through information design / 情報設計による社会接続

---

## Research Perspective / 研究視点

This research observes how a hotel can function not only as accommodation, but also as a cultural mediation space connecting people, community, emotion, and regional culture.

本研究では、観光施設が単なる宿泊空間ではなく、人・感情・地域文化を接続する文化的媒介空間として機能する可能性を観察・記録している。
